# Dataset: WWTP (Wastewater Treatment Plant) Flows

Dataset with WWTP inflows

## Usage

``` r
wwtp
```

## Format

A tibble with 1,152 rows and three columns

- date:

  date

- id:

  id of WWTP

- cbm_per_second:

  measured flow in cubicmeter per second, backcalculated from monthly
  effluent sums by dividing through number of days in month
