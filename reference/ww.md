# Dataset: Waterworks (WW) Abstractions per Gallery

Dataset with waterworks abstractions per gallery

## Usage

``` r
ww
```

## Format

A tibble with 17,141 rows and three columns

- date:

  date

- id:

  id of WW gallery

- cbm_per_second:

  measured flow in cubicmeter per second, backcalculated from monthly
  abstraction sums by dividing through number of days in month
