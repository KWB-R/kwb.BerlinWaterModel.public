# Add scenario

Add scenario

## Usage

``` r
add_scenario(config, use_scenario = FALSE, debug = TRUE)
```

## Arguments

- config:

  config as retrieved by
  [`config_read`](https://kwb-r.github.io/kwb.BerlinWaterModel.public/reference/config_read.md)

- use_scenario:

  use scenario (default: FALSE)

- debug:

  print debug messages (default: TRUE)

## Value

list with input parameters required for function
[`prepare_input`](https://kwb-r.github.io/kwb.BerlinWaterModel.public/reference/prepare_input.md)
