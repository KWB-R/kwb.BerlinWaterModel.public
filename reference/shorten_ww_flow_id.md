# Helper: shorten WW flow id

Helper: shorten WW flow id

## Usage

``` r
shorten_ww_flow_id(df, col_flow_id = "flow_id")
```

## Arguments

- df:

  df with WW flow data andf flow ids as columns

- col_flow_id:

  column name with flow id (default: "flow_id")

## Value

vector with shortened WW flow ids
