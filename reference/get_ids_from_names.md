# Helper function: get ids from names

Helper function: get ids from names

## Usage

``` r
get_ids_from_names(vector, config)
```

## Arguments

- vector:

  vector with names

- config:

  config as retrieved by
  [`config_read`](https://kwb-r.github.io/kwb.BerlinWaterModel.public/reference/config_read.md)

## Value

tibble with section names and ids contained in configuration
