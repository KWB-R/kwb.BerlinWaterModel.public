# Plot Network: simple

Plot Network: simple

## Usage

``` r
plot_network_simple(network)
```

## Arguments

- network:

  tibble with water cycle flow network data, as retrieved by
  `{`[`prepare_network`](https://kwb-r.github.io/kwb.BerlinWaterModel.public/reference/prepare_network.md)`}`

## Value

simple network graph
