# Check Network Errors

Check Network Errors

## Usage

``` r
check_network_errors(network)
```

## Arguments

- network:

  tibble with water cycle flow network network, as retrieved by
  `{`[`prepare_network`](https://kwb-r.github.io/kwb.BerlinWaterModel.public/reference/prepare_network.md)`}`

## Value

list with information on network errors
